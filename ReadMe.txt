EN -
 Project : genetique
 Version : 1.0
 Author  : Parlons Archi !
 See     : https://www.youtube.com/@christophecadet
 
     This project proposes a generic implementation of genetic algorithms in Java 11 for didactic purposes.

	 The concepts related to genetic algorithms are explained here: https://www.youtube.com/watch?v=yN4_uylPGu8       
	 The available code is explained here: https://www.youtube.com/watch?v=oft-Xnr9Xp0     
	 
     The "generic" package contains the generic modules of the solution.
     the "alphabetic" package contains an example of use looking for a mystery word.      
     the "numeric" package contains an example of use looking for a mystery number.

     Ce projet propose une implémentation générique d'algorithme génétique en Java 11 dans un but didactique.

     le package "alphabetic" comporte un exemple d'utilisation recherchant un mot mystere.      
     le package "numeric" comporte un exemple d'utilisation recherchant un nombre mystere.
